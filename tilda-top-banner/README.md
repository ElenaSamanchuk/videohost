# Верхняя рекламная плашка для Tilda Zero Block

Плашка появляется сверху, висит ~5 секунд, уезжает вверх. Можно закрыть крестиком. Сама плашка ведёт по ссылке (как настроено в Zero Block).

## Быстрая установка

### 1. Zero Block

1. Создайте Zero Block с дизайном плашки.
2. На **весь контент** (кнопка/группа/фон) повесьте ссылку — редирект при клике.
3. **Крестик** — отдельный элемент **без ссылки**. В настройках элемента → **CSS class name** укажите:

   ```
   top-banner-close
   ```

   Крестик лучше вынести поверх контента (выше по слоям в Zero Block).

4. Запомните ID блока: в предпросмотре в HTML будет `#rec123456789` — нужны только цифры.

### 2. Подключение кода

**Важно: нужны и CSS, и JS.** Без CSS классы `top-banner--visible` / `top-banner--hiding` ничего не делают.

**Проще всего** — один блок T123 (HTML) на странице: скопируйте целиком `kochfit-snippet.html` (или `<style>` + `<script>` из него).

**Или раздельно:**

**CSS** — Настройки сайта → **Ещё** → **CSS для всего сайта**  
(или HTML-блок T123 / код в `<head>`):

Скопируйте `banner.css`, замените `123456789` на ваш ID.

**JS** — второй блок T123 или **HTML-код для вставки внутрь BODY**:

Скопируйте `banner.js`, замените `REC_ID` на ваш ID.

### 3. Настройки в banner.js

| Переменная | По умолчанию | Описание |
|---|---|---|
| `REC_ID` | `'123456789'` | ID Zero Block |
| `SHOW_DURATION` | `5000` | Время показа, мс |
| `CLOSE_SELECTOR` | `.top-banner-close` | Класс крестика |

## Поведение

- `position: fixed` — плашка всегда сверху при скролле.
- `z-index: 1000001` — поверх контента, куки, стрелки «вверх» и чата.
- Клик по плашке → переход по ссылке Zero Block.
- Клик по крестику → закрытие без перехода.
- Через 5 сек → плавное скрытие вверх.

## Если крестик внутри ссылки

В Zero Block **не оборачивайте крестик в ту же ссылку**, что и баннер.  
Структура:

```
[Фон / группа со ссылкой — весь баннер]
[Крестик .top-banner-close — без ссылки, поверх]
```

## Не показывать повторно после закрытия (опционально)

Добавьте в `hideBanner()` перед `isHidden = true`:

```javascript
try { sessionStorage.setItem('topBannerClosed', '1'); } catch (e) {}
```

И в начале скрипта после `if (!rec) return;`:

```javascript
try {
  if (sessionStorage.getItem('topBannerClosed') === '1') return;
} catch (e) {}
```
